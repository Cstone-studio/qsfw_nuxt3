# gsfw_vue_front
基于vue的前端基盘

集成内容包括：  
1.vue2.x  
2.nuxt.js  
3.typeScript  
4.element-ui  
5.vuex  
6.axios  
7.axios http返回码封装  
8.dotenv多环境变量配置  
9.配置代理 解决跨域问题  
10.echart图表  
11.vue auth  
12.基于jwt登陆  
13.ESLint + Prettier代码风格与格式检验  
14.移动ui框架vant  

# Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
****
# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
